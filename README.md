# EVENT2U
Resolução do trabalho de APSOO sobre a aplicação hibrída (com elementos desktop e web) EVENT2U 

===========================================================================

**Como rodar a aplicação:**

1. Baixe os requirements:
(no terminal escreva)
--> pip install -r requirements.txt

2. Rode a aplicação: 
--> python Projeto/run.py

3. Aplicação aberta!
